## 项目使用三方库

1、Loggie:动画库，之前在AndroidView中使用很多
2、Navigation:页面之间的跳转
3、Navigation-animation：配合Navigation进行使用，可添加动画
4、Accompanist-pager:Compose中的ViewPager
5、Room:Jetpack数据库
6、Splashscreen:启动动画
7、Hilt:依赖注入
